# 第二引力销售系统 V3 - 重建精简版

这是对原项目的低风险重建版。页面结构、localStorage 数据字段、Supabase `sync_store` 同步结构和主要全局函数名保持兼容；冗余历史补丁被归并到单一入口脚本。

## 本次重建结果

- 原项目：33 个文件，22 个 JS 文件，约 3396 行 JS。
- 新项目：9 个文件，2 个 JS 文件（含 `sw.js`），约 2433 行 JS。
- 页面只加载一个业务脚本：`assets/js/app.bundle.js`。
- 已移除被新版自动同步覆盖的旧云同步补丁：`72`、`73`、`74`、`75`、`76`、`77`。
- 保留最新有效能力：PWA 移动收银、用户稳定性修复、自动云同步 v9、扫码枪输入、移动端紧凑布局、管理员清空销售记录、Toast/同步提示。

## 部署

把本目录整体上传到 HTTPS 静态站点即可，例如 Cloudflare Pages、Netlify、Vercel、GitHub Pages 或 Nginx。

首次上线后建议：

1. 清理浏览器旧 Service Worker 缓存，或等待新版 `3.2.0-rebuilt` 自动接管。
2. 登录后立即修改默认管理员密码。
3. 检查 Supabase RLS 和 `sync_store` 权限。
4. 在移动端添加到主屏幕并测试扫码、结账、报表和同步。
